Privacy Policy

Thoughtfully keeps personal details focused and intentional.

Thoughtfully is designed to help users save birthdays, gift ideas, favorites, personal notes, and meaningful moments.

Information you save

Information entered into the app is stored locally on the user's device unless otherwise stated.

Notifications

Notifications are optional and are used for birthday reminders, optional prep reminders, and gentle app reminders.

Photos

Photo access is optional and is used only if a user chooses to add a profile photo.

Contacts

Contacts access is optional and is used only when a user chooses to import one contact. Thoughtfully imports only available name, birthday, photo, and notes for the selected contact. The app does not import phone numbers, email addresses, addresses, company names, or job titles.

Experimental idea features

Thoughtfully may offer experimental idea features that help generate birthday or gift suggestions from details saved on a profile. These features can be turned off in Settings.
